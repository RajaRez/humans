This is an open poem.

The main text of the work is in the README.md file and should stay in that location.

Part of this work is in the revision history, watching it as it changes. An evolving record of my continual reckoning with what it means to be between quantifiable and non-quantifiable.

Please feel free to:

- Fork the repository
- Make your own revision
- Distribute
- Submit issues and pull requests
